# Change Log

This is the change log for DelphiDabbler _BogBat_.

All notable changes to this project are documented in this file. Releases are listed in reverse version number order.

Release v0.2.0 of 2026-06-26

* Fixed errors in help screen text [issue #3].
* Fixed bug where help, version and usage text were being written to stderr instead of stdout [issue #1].
* Changed to compile with Delphi 13.1 instead of Delphi 12.2.
* `Deploy.bat` now reads the program's release version number from `src/VERSION` instead of from a command line parameter [issue #4].

Release v0.1.0 of 2025-06-05

* Initial release.
